# Readme Generator  

## User Story 

-AS A developer  
-I WANT a README generator  
-SO THAT I can quickly create a professional README for a new project  

## Description 

-Node.js program to generate a readme file from the command line  

## Built With 

-Javascript  
-Node.js  
-npm inquirer  

## Installation  

`npm init`
  
`npm install inquirer`
